# Application Architecture
## Overview
The application is built using Next.js and React, with a focus on scalability and maintainability.
## Components
* TodoList: a sortable list of todo items
* TodoItem: a single todo item with drag handle
## Storage
* Local storage is used to persist todo items between sessions.
## Technical Details
The application utilizes a strict TypeScript approach, with functional components and custom hooks for state management. The architecture is designed with clean code principles in mind, ensuring readability and maintainability.
